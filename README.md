## 🚀 1단계: 최초 설치 및 실행

### 1-0. 사전 준비 (한 번만 하면 됨)

이 프로그램을 쓰려면 **Node.js**라는 프로그램이 컴퓨터에 설치되어 있어야 합니다.

1. 인터넷 브라우저(크롬 등)에서 아래 주소로 이동합니다:
   👉 **https://nodejs.org**
2. 화면에 초록색 버튼 2개가 보입니다. **왼쪽 버튼 (LTS)** 을 클릭해서 다운로드합니다.
3. 다운로드된 파일을 **더블클릭**해서 설치합니다.
   - 설치 중 나오는 화면에서는 모두 **Next → Next → Install → Finish** 누르면 됩니다.
   - 아무 설정도 바꿀 필요 없습니다.
4. 설치가 끝나면 컴퓨터를 한 번 **재시작**합니다.

### 1-1. 프로그램 실행하기 (매번 사용할 때마다)

> **"터미널"** 이란? 컴퓨터에 글자로 명령을 내리는 검은 화면입니다. 어렵지 않아요!

**방법 A — 가장 쉬운 방법 (폴더에서 바로 열기)**

1. 바탕화면에 있는 **대본자동화** 폴더를 찾아서 **더블클릭**으로 엽니다.
2. 폴더 안의 **빈 공간**에서 **마우스 오른쪽 클릭**합니다.
3. 메뉴에서 **"터미널에서 열기"** 를 클릭합니다.
   _(또는 "여기서 PowerShell 열기" 라고 적혀 있을 수도 있어요)_
4. 검은색(또는 파란색) 창이 뜹니다. 이게 터미널입니다!

**방법 B — 키보드 단축키로 열기**

1. 키보드에서 `Windows키(⊞)` + `R` 을 동시에 누릅니다.
2. 작은 창이 뜨면 `cmd` 라고 입력하고 **확인** 클릭합니다.
3. 검은 창이 뜨면, 아래 글자를 **그대로** 입력하고 **Enter** 키를 누릅니다:
   ```
   cd %USERPROFILE%\Desktop\대본자동화
   ```

**이제 프로그램을 시작합니다!**

5. 터미널 창에 아래 글자를 **그대로** 입력하고 **Enter** 키를 누릅니다:
   ```
   npm run dev
   ```
6. 잠시 기다리면(5~10초) 아래와 같은 글자가 나타납니다:
   ```
   ✅ Database initialized
   🚀 서버 실행 중: http://localhost:3001
   VITE ready → http://localhost:5173
   ```
   이 글자가 보이면 **성공**입니다! 🎉

> [!CAUTION]
> ❌ 만약 `npm은(는) 인식할 수 없는 명령입니다` 라는 빨간 글자가 나오면
> → 1-0단계의 Node.js 설치를 다시 해주세요. 설치 후 컴퓨터 재시작도 꼭 해주세요.

> [!IMPORTANT]
> ⚠️ 이 터미널 창은 프로그램을 쓰는 동안 **절대 닫지 마세요!**
> 닫으면 프로그램이 꺼집니다. 최소화(─ 버튼)는 해도 됩니다.

### 1-2. 웹 앱 접속하기

1. 크롬, 엣지 등 **인터넷 브라우저**를 엽니다.
2. 주소창에 아래를 입력하고 **Enter**:
   ```
   localhost:5173
   ```
3. YouTube 주제 분석기 화면이 나타나면 성공! 🎉

## 🔑 2단계: API 키 설정 (최초 1회)

> [!IMPORTANT]
> API 키가 없으면 떡상 검색, 영상 수집, AI 분석 기능을 사용할 수 없습니다.

### 2-1. YouTube API 키 발급
1. [Google Cloud Console](https://console.cloud.google.com/) 접속 → 로그인
2. **프로젝트 만들기** → 프로젝트 이름 입력 → 만들기
3. 좌측 메뉴 → **API 및 서비스** → **라이브러리**
4. **"YouTube Data API v3"** 검색 → **사용 설정**
5. 좌측 **사용자 인증 정보** → **+ 사용자 인증 정보 만들기** → **API 키**
6. 생성된 키 복사

### 2-2. Gemini API 키 발급
1. [Google AI Studio](https://aistudio.google.com/) 접속
2. 좌측 **API keys** → **Create API key**
3. 생성된 키 복사

### 2-3. API 키 입력
1. 웹 앱 → 사이드바 **⚙️ 설정** 클릭
2. YouTube API 키 입력 → **저장**
3. Gemini API 키 입력 → **저장**
4. `✅ 설정됨` 표시 확인

## 🎭 3단계: 장르 프리셋 선택

> 분석의 정확도를 높이기 위해 장르에 맞는 카테고리를 설정합니다.

1. **⚙️ 설정** → 우측 **🎭 장르 프리셋**
2. 본인 채널 장르에 맞는 프리셋 클릭:
   - **야담/역사** — 야담, 사극, 역사 채널
   - **괴담/호러** — 공포, 미스터리 채널
   - **요리/먹방** — 먹방, 레시피 채널
   - **교육/지식** — 강의, 정보 채널
   - **게임** — 게임 리뷰, 실황 채널
3. "프리셋 적용 완료!" 토스트 확인

## 🔥 4단계: 떡상 영상 검색 (핵심 기능)

> 키워드만 넣으면 지금 뜨고 있는 영상을 S/A/B/C/D 등급으로 찾아냅니다.

1. 사이드바 → **🔥 떡상 검색** 클릭
2. 검색 조건 입력:
   - **키워드**: 예) `노후 사연`, `괴담 실화`, `자취 요리`
   - **기간**: 최근 1주일 / 1개월 / 3개월 / 전체
   - **영상 길이**: 숏폼(~4분) / 미디엄(4~20분) / 롱폼(20분+)
   - **최소 구독자**: 너무 작은 채널 제외 (추천: 100~1000)
   - **최소 조회수**: 너무 적은 영상 제외 (추천: 1000~10000)
3. **🔍 검색** 클릭
4. 결과에서 확인:
   - **S등급** 🔴 = 구독자 대비 조회수 2000%+ (대박)
   - **A등급** 🟠 = 500~2000% (떡상)
   - **B등급** 🟡 = 100~500% (준수)
   - **C등급** 🟢 = 30~100% (보통)
   - **D등급** ⚪ = 30% 미만

### 떡상 검색 결과 활용

| 버튼 | 기능 |
|------|------|
| **▶ 보기** | YouTube에서 영상 열기 |
| **📥 수집** | 내 DB에 영상 저장 (이후 분석 가능) |
| **📥 CSV** | 검색 결과 엑셀로 다운로드 |
| **제목 필터** | 결과 내에서 특정 키워드만 필터링 |
| **정렬** | 떡상지표순 / 조회수순 / 참여율순 / 최신순 |

## 📺 5단계: 채널 등록 & 영상 수집

> 벤치마킹할 채널을 등록하고 영상을 자동 수집합니다.

### 5-1. 채널 등록
1. 사이드바 → **📺 채널 관리** 클릭
2. **+ 채널 추가** 버튼
3. 채널 정보 입력 (3가지 중 하나):
   - 채널 URL: `https://www.youtube.com/@채널명`
   - 채널 ID: `UC...` 로 시작하는 ID
   - 핸들: `@채널명`
4. **미리보기** → 채널 정보 확인 → **등록**

### 5-2. 영상 수집
1. 등록된 채널 카드의 **🔄 영상 수집** 클릭
2. 진행률 표시 → 완료까지 대기 (채널 규모에 따라 1~5분)
3. 수집 완료 후 자동으로:
   - ✅ 영상 정보 저장 (제목, 조회수, 업로드일 등)
   - ✅ 자막 추출 (설정에서 ON인 경우)
   - ✅ AI 키워드 추출
   - ✅ AI 카테고리 분류
   - ✅ 자막 요약

## 🎬 6단계: 영상 분석 (스크립트·댓글·벤치마킹)

> 수집된 영상을 다양한 방식으로 분석합니다.

### 뷰 모드 전환
- **☰ 리스트뷰**: 테이블 형태 (데이터 한눈에)
- **▦ 카드뷰**: 썸네일+정보 카드 (시각적)

### 📜 스크립트 보기
1. 영상의 **📜** 버튼 클릭
2. 자막 전문이 모달로 표시
3. **📋 복사** → 클립보드 복사 (ChatGPT에 붙여넣기용)
4. **📥 TXT 다운로드** → 파일로 저장

### 💬 댓글 AI 분석
1. 영상의 **💬** 버튼 클릭
2. YouTube에서 좋아요 순 상위 150개 댓글 자동 수집
3. Gemini AI가 자동 분석:
   - **감정 비율**: 긍정/중립/부정 %
   - **시청자 공감 포인트 TOP 5**: 사람들이 가장 반응한 부분
   - **새 주제 추천**: 이 댓글 반응을 활용한 영상 아이디어
4. **📥 댓글 CSV** → 댓글 엑셀 다운로드

### 📊 AI 벤치마킹 리포트 (핵심 기능)
1. 영상의 **📊** 버튼 클릭
2. 10~20초 대기 (댓글 수집 + AI 분석)
3. 자동 생성 리포트:
   - **떡상 등급 & 참여율** 요약
   - **왜 이 영상이 떴는가** 4가지 이유
   - **제목 분석**: 어떤 전략이 효과적이었는지
   - **스크립트 구조** 분석 (도입/전개/절정/결말)
   - **벤치마킹 포인트**: 따라 해야 할 것 3가지
   - **새 주제 제안**: 이 영상을 벤치마킹한 주제 3개
   - **주의사항**: 리스크 팩터

## 🔍 7단계: 내 주제 유사도 확인

> 새 영상 아이디어가 기존 영상과 얼마나 겹치는지 확인합니다.

1. 사이드바 → **🔍 주제 비교** 클릭
2. 3가지 입력 모드 중 선택:
   - **📝 새 아이디어** — 제목+설명 직접 입력
   - **🔗 YouTube URL** — 기존 영상 URL 입력
   - **🏷️ 키워드** — 핵심 키워드만 입력
3. **비교 분석** 클릭
4. 결과 확인:
   - **게이지**: 0~100% 유사도
   - 🟢 0~30% = **안전** (새로운 주제!)
   - 🟡 30~60% = **주의** (일부 겹침)
   - 🔴 60~100% = **위험** (많이 겹침)
   - **TOP 10 유사 영상** 목록

3. **📊 분석** 클릭

   - 🟢 **0~2개** = 아직 아무도 안 다룬 영역 → **기회!**
   - 🟡 **소수** = 경쟁 적음 → **좋은 기회**
   - 🔴 **많음** = 이미 포화 → **피하기**
5. AI 추천 주제 확인

## 🧩 크롬 확장 프로그램 (선택)

> YouTube 시청 중 바로 수집/비교를 할 수 있습니다.

### 설치 방법
1. Chrome → 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. `대본자동화/chrome-extension` 폴더 선택
5. 확장 프로그램 목록에 **YouTube 주제 분석기** 표시 확인

### 사용 방법
- **영상 페이지**: 제목 옆에 **🎬 수집** / **🔍 비교** 버튼 표시
- **채널 페이지**: **📺 이 채널 등록** 버튼 표시
- **팝업**: 확장 아이콘 클릭 → 서버 상태·DB 통계 확인

## ❓ 자주 묻는 질문

**Q: "YouTube API 키가 설정되지 않았습니다" 오류가 나요.**
→ ⚙️ 설정 → YouTube API 키를 입력하고 저장했는지 확인. 복사할 때 앞뒤 공백 주의.

**Q: 떡상 검색이 안 돼요.**
→ YouTube Data API v3가 활성화되었는지 Google Cloud Console에서 확인.

**Q: AI 벤치마킹 리포트가 생성되지 않아요.**
→ Gemini API 키가 올바른지 확인. Google AI Studio에서 키를 새로 발급해보세요.

**Q: 자막(스크립트)이 안 나와요.**
→ 해당 영상에 자막이 없거나, 자막이 비공개 설정된 영상입니다.

**Q: 나중에 다시 시작하려면?**
→ 터미널에서 `cd 대본자동화` → `npm run dev` 실행 후 http://localhost:5173 접속.
모든 데이터는 `data/yadam.db`에 자동 저장되므로 유지됩니다.
